# Dniester Commission — Static Prototype (Jinja2 + Vite + SCSS)

This project renders HTML pages from Jinja2 templates (Python) and bundles CSS/JS with Vite (Node).
Output is a static site in `dist/`.

## Prerequisites
- Python 3.10+
- Node.js 18+ (LTS recommended)

## Install
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt

npm install
```

## Build (one command)
```bash
npm run build
```
This will:
1) render pages to `dist/*.html` (Jinja2)
2) build assets to `dist/assets/main.css` and `dist/assets/main.js` (Vite)

## Images
Place your images in:
```
dist/images/
```
or add a `src/static/images/` folder and they will be copied to `dist/` during `python build.py`.

## Dev
- For assets (SCSS/JS):
  ```bash
  npm run dev
  ```
  Note: HTML is still generated by Jinja2. Re-run `python build.py` when templates change.

## Project layout
- `src/templates/` — base layout + partials
- `src/pages/` — page templates (extend base)
- `web/` — Vite entrypoints (`main.js`, `styles.scss`, `legacy.scss`)
- `dist/` — built site output

## Development workflow (SCSS + local Bootstrap)

Install once:

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt

npm install
```

Build production `dist/` (HTML + assets):

```bash
npm run build
```

Dev mode (rebuild Jinja pages + rebuild assets on change + serve `dist/`):

```bash
npm run dev:all
```

Then open:
- http://localhost:8000/

Note: This dev setup rebuilds files on change; your browser may still require a manual refresh.
